Introduction to CopAssist

CopAssist is an AI-powered platform designed to assist with law enforcement documentation, specifically First Information Reports (FIRs). Our platform helps users generate legally compliant documents, organize case files, and access relevant legal information.

This documentation will guide you through setting up and using CopAssist, from basic usage to advanced customization and integration.

Note: CopAssist is designed to assist with legal documentation but is not a replacement for professional legal advice. Always consult with a qualified legal professional for specific legal matters.

FIR Basics

A First Information Report (FIR) is an important legal document that initiates the criminal justice process. CopAssist helps you create legally valid FIRs by guiding you through the required information and formatting.

Components of an FIR

  • Date and Time: When the incident occurred
  • Location: Where the incident took place
  • Nature of Incident: Type of incident (theft, assault, etc.)
  • Description: Detailed account of the incident
  • Parties Involved: Information about complainant and accused
  • Witnesses: Details of any witnesses
  • Evidence: Description of available evidence

Using the FIR Generator

CopAssist's FIR generator walks you through each required field, providing guidance and validation to ensure your document meets legal requirements.

API Authentication

To use the CopAssist API, you'll need to authenticate your requests using an API key. This section explains how to obtain and use your API key securely.

Obtaining an API Key

  1. Log in to your CopAssist account
  2. Navigate to Account Settings > API
  3. Click "Generate API Key"
  4. Store your API key securely - it will only be shown once

Using Your API Key

Include your API key in the header of each request:

// Example API request curl -X POST https://api.copassist.com/v1/fir/generate \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "incident_type": "theft", "date": "2025-03-09T14:30:00", "location": "123 Main Street, Apartment 4B", "description": "Laptop and wallet stolen from residence" }'
